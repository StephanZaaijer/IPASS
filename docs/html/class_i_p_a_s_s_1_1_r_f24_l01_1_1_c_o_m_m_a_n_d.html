Detailed Description

SUBCLASS COMMAND.

SUBCLASS COMMAND that contains only out of static const uint8_t with the addresses of the RF24L01-registers

Warning
the commands are just mask and because of that they can not be used without some change so read datasheet before calling these commands you can crash your program by executing command without knowing what is does.
