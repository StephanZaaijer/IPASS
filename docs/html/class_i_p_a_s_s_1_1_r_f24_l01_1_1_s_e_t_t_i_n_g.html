◆ CD

RF24L01::SETTING::Setting IPASS::RF24L01::SETTING::CD = {RF24L01::REGISTER::CD, 0x01, 0}
static

static Settings that contains the Register_name and the bitmask of the CD-setting

The CD setting is a setting which is check for a Carried detect if HIGH the device is in a jam situation which means the channel should be changed. if LOW the the device is either out-of-range or jammed by broadband signals like WLAN), the device can stay on the same channel but should perform a dummy write to RF_CH to reset the PLOS_CNT

Note
Only recommended to use if the TX device does not succeed to get packets through
